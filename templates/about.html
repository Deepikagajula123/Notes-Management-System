About Notes Management System

The Notes Management System is a simple web application that helps users create, organize, and manage their personal notes easily.

Users can register and log in securely to access their dashboard. From the dashboard, they can add new notes, view all saved notes, update existing notes, or delete notes they no longer need.

This system also includes a user profile page where users can upload a profile picture and view their account information.

The application is built using Flask, Python, MySQL, HTML, and CSS, providing a clean and responsive user interface.
